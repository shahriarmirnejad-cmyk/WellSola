WellSola is a proactive AI companion application designed to support emotional well-being among older adults. The system combines gentle check-ins, medication management, emotionally supportive conversation, and assisted communication in a single accessible mobile interface.

The application is built around four core ethical principles drawn from the original proposal:
•	Supporting user autonomy — all AI actions are suggestions, never commands
•	Avoiding emotional dependency — Solla consistently nudges users toward real human relationships
•	Transparency — data sharing with caregivers and family is always visible to the user
•	Accessibility — large text, generous tap targets, voice input, and simple language throughout
